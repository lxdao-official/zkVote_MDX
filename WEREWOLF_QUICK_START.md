# 狼人杀多人联机 - 快速开始指南

## 🎮 如何开始游戏

### 方式一：创建房间（房主）

1. 点击主菜单的 **"Play ZK Werewolf"** 按钮
2. 在游戏模式选择页面，点击 **"Multiplayer"**（多人模式）
3. 点击 **"Create New Room"**（创建新房间）
4. 设置游戏参数：
   - **Total Players（总玩家数）**: 4-12人（用滑块调整）
   - **AI Bots（AI机器人数）**: 0到总玩家数-1（用滑块调整）
   - 真人玩家席位 = 总玩家数 - AI机器人数
5. 点击 **"Create Room"**
6. 页面会显示一个**邀请链接**，点击 **"Copy"** 按钮复制
7. 通过微信、QQ等方式分享链接给朋友
8. 等待朋友加入并准备好
9. 所有人准备后，点击 **"Start Game"** 开始游戏

### 方式二：加入房间（玩家）

**通过链接加入（推荐）：**
1. 点击朋友分享的链接
2. 会自动跳转到加入房间页面
3. 输入你的名字
4. 连接钱包
5. 点击 "Generate ZK Identity"
6. 自动加入房间

**手动输入房间ID：**
1. 点击 **"Join Existing Room"**
2. 输入房主提供的房间ID（长字符串）
3. 输入你的名字
4. 连接钱包并生成ZK身份
5. 点击 **"Join Room"**

## 🎯 游戏流程

### 1. 大厅等待
- 玩家陆续加入房间
- 每个玩家点击 **"Ready"** 按钮表示准备
- 房主等待所有人准备就绪
- 房主点击 **"Start Game"** 开始

### 2. 角色分配
- 系统自动随机分配角色
- 每个玩家只能看到自己的角色
- 角色包括：
  - 🐺 **狼人（Werewolf）**: 晚上杀人
  - 🔮 **预言家（Seer）**: 晚上查验一人身份
  - 💊 **女巫（Witch）**: 有解药和毒药
  - 👨‍🌾 **村民（Villager）**: 白天投票

### 3. 夜晚阶段
- **狼人行动**: 选择一个玩家攻击
- **预言家行动**: 选择一个玩家查验身份
- **女巫行动**: 决定是否使用解药或毒药
- **村民**: 睡觉等待天亮

### 4. 白天讨论
- 公布夜晚死亡信息
- AI机器人会自动发言
- 真人玩家可以在聊天框中交流
- 讨论时间结束后进入投票

### 5. 白天投票
- 每人投票驱逐一个可疑玩家
- 票数最多的玩家出局
- 如果平票则无人出局

### 6. 胜利条件
- **好人胜利**: 所有狼人出局
- **狼人胜利**: 狼人数量 ≥ 好人数量

## 💡 游戏技巧

### 对于狼人：
- 伪装成村民或其他角色
- 不要太早暴露身份
- 优先击杀有威胁的角色（预言家、女巫）
- 和队友配合混淆视听

### 对于预言家：
- 谨慎公开身份
- 记录查验结果
- 引导好人投票狼人
- 注意保护自己

### 对于女巫：
- 解药和毒药各只有一次使用机会
- 第一晚通常会救人
- 记住被杀的玩家，推理狼人身份
- 谨慎使用毒药

### 对于村民：
- 仔细观察每个人的发言
- 寻找逻辑矛盾
- 相信预言家的查验结果
- 积极参与投票

## ❓ 常见问题

### Q: 连接失败怎么办？
A: 检查网络连接，刷新页面重试。确保浏览器允许WebRTC。

### Q: 房主离开了怎么办？
A: 目前房主离开游戏会中断，建议房主保持在线。

### Q: 可以中途加入吗？
A: 不可以，必须在游戏开始前加入。

### Q: AI机器人会作弊吗？
A: 不会，AI遵循相同规则，不知道其他玩家角色。

### Q: 聊天记录会保存吗？
A: 不会，仅在当前会话中存在。

### Q: 游戏卡住了怎么办？
A: 刷新页面，房主可以重新创建房间。

## 🔧 技术说明

- 使用 **PeerJS** 实现 P2P 通信
- 基于 **WebRTC** 技术，无需中心服务器
- 使用 **Semaphore** 协议的零知识证明
- 完全去中心化，数据在玩家间直接传输
