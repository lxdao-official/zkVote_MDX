# 测试系统快速使用指南

## 用户操作步骤

### 1️⃣ 连接钱包
- 点击页面右上角的"Connect Wallet"按钮
- 使用 MetaMask 连接你的钱包

### 2️⃣ 开始测试
- 在评价模块上方找到 **"📝 开始测试"** 按钮
- 点击按钮打开测试弹窗

### 3️⃣ 答题
- 共有 15 道题：10 道单选 + 5 道多选
- **单选题**：点击选项即可选中
- **多选题**：可以勾选多个选项
- 完成所有题目后才能提交

### 4️⃣ 提交查看结果
- 点击"提交答案"按钮
- 系统自动评分并显示：
  - 总分（百分制）
  - 正确/错误题目数量
  - 每道题的详细解析

### 5️⃣ 查看答题详情
- 滚动查看每道题的正确答案
- 绿色标记 ✓ = 回答正确
- 红色标记 ✗ = 回答错误
- 阅读每题的解析说明

### 6️⃣ 选择操作
- **重新测试**：清空答案重新答题
- **关闭**：关闭测试窗口

## 评分标准

| 分数范围 | 评价 | 说明 |
|---------|------|------|
| 90-100分 | 🎉 优秀 | 完全掌握了ZK投票核心概念 |
| 70-89分 | 👍 良好 | 对ZK投票有较好理解 |
| 60-69分 | 💪 及格 | 继续加油，多复习教程 |
| 0-59分 | 📚 需加强 | 建议再仔细阅读教程 |

## 注意事项

⚠️ **必须先连接钱包**才能进行测试  
⚠️ 所有题目都必须作答才能提交  
⚠️ 多选题必须选择所有正确选项（少选或多选都不得分）  
✅ 可以重复测试，不限次数  
✅ 测试结果会关联到你的钱包地址  

## 题目范围

测试题目基于 MyFirstZKVote 教程内容，涵盖：
- ✓ 区块链投票的优势与隐私问题
- ✓ 零知识证明的基本概念和三大性质
- ✓ Identity Commitment、Nullifier、Merkle 树
- ✓ ZK投票的完整流程
- ✓ zk-SNARK 和 zk-STARK 的区别
- ✓ 实际应用中的安全考虑

## 技术支持

如遇到问题，请检查：
1. 是否已连接钱包
2. 浏览器控制台是否有错误信息
3. 网络连接是否正常
